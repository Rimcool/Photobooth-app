# 📸 Photobooth App

A desktop photobooth application built with Electron.

## ✨ Features
- Live Camera Preview
- 3…2…1 Countdown
- Filters (B&W, Sepia, Pink)
- Frames (Eid, Wedding, Birthday, Ramadan)
- Stickers (PNG overlays)
- Save photos locally
- Photo Gallery
- Print support (coming soon)

## 🛠 Tech Stack
- Electron
- HTML
- CSS
- JavaScript
- Canvas API
- MediaDevices API

## 🚀 How to Run

```bash
npm install
npm start
