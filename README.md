# 📸 Cute Snaps Photobooth 🌼

**Your super adorable desktop webcam photobooth**  
Perfect for selfies, parties, Eid vibes, weddings, birthdays, Ramadan nights,  
or just spreading good energy with friends! 💕

Turn your camera into a magical kawaii photo experience  
with filters, stickers, frames, countdowns, and instant photo strips! ✨

## ✨ Features

- 🎥 Live camera preview with real-time filters  
- ⏱️ 3…2…1… Smileeee! countdown before each snap  
- 📸 Auto-captures 4 photos → creates a cute photo strip  
- 🌈 Filters: None, B&W, Sepia, Pink tone (live + saved in photo)  
- 🖼️ Frames: Eid moon, Wedding roses, Birthday balloons, Ramadan lanterns  
  (drop your own transparent PNGs)  
- 🎀 Stickers: crown 👑, bunny 🐰, glasses 🕶️, bow 🎀, mustache & more  
  (click to place in center)  
- 🖼️ Gallery screen showing your 4-photo strip  
- 💾 Save photos locally (/photos folder)  
- 📤 Share – copy image to clipboard (paste into WhatsApp, Instagram, etc.)  
- 🖨️ Print button – send directly to printer  
- 🔄 Retake anytime for more fun  
- Kawaii design: pastel pinks, checkerboards, daisies 🌼, smileys 😊, Comic Neue font

## 🚀 Quick Start

1. Clone the repo  
   ```bash
   git clone https://github.com/Rimcool/Photobooth-app.git
   cd Photobooth-app

Install dependenciesBashnpm install
Run the appBashnpm start

Allow camera access → start snapping! 📸💕
🛠️ Build Installer (.exe / .dmg)
Bashnpm run build
Files appear in the dist/ folder.
🎨 Customize

Add frames & stickers: drop transparent PNGs into
assets/frames/ and assets/stickers/
Change colors, fonts, backgrounds in HTML & CSS files
Want draggable stickers or more features? Open an issue!
